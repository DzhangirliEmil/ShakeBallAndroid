# ShakeBallAndroid

## Описание приложения
Основная функция приложения - "Шар-гадалка". Пользователю предоставлен шар, который реагирует на тряску телефона. После окончания тряски пользователю показывается один из нескольких советов из списка и картинка, иллюстрирующая этот совет. Можно будет выбрать один из нескольких предустановленных списков или же самому добавить свой список. Пользователю будет предоставлена возможность самому создавать дизайн новых шаров, а так же управлять их свойствами (декремент затухания колебаний шарика, прыгучесть шарика, чувствительность к тряске и сила вибрации).

### Ссылка на демо-версию приложения https://play.google.com/store/apps/details?id=emil.international.StayHomeBall&hl=ru&gl=US

## Реализовано на данный момент

На данный коммит в приложении две активности: главный экран и экран настроек.

### Главный экран приложения (MainActivity)

![image](https://user-images.githubusercontent.com/79907936/110104414-2dc0da00-7db8-11eb-9352-eba9752441b2.png)

Главный экран представляет из себя активность с интерфейсом выбора отображаемого фрагмента при помощи BottomNavigationView. Всего у пользователя есть возможность выбрать один из трёх фрагментов - фрагмент просмотра листа советов шарика, фрагмент с шариком или фрагмент с листом созданных шаров.

На данный момент реализован фрагмент с подвижным шариком. 



#### Фрагмент с шариком (BallFragment)
![image](https://user-images.githubusercontent.com/79907936/110104598-682a7700-7db8-11eb-80cc-5a4c0196aa3d.png)

При тряске телефона трясется шарик. При помощи SpringAnimation шарик приделан к центру экрана. При тряске совет и картинка к совету внутри шарика пропадает. При окончании анимации фрагмент запрашивает из вызвавшего его класса текст следующего совета и ID картинки к нему.
Для использования фрагмента в классе необходимо реализовать интерфейс NextValue с методами setBallParameters() и getAdviceText()

### Активность настройки параметров шарика.
![image](https://user-images.githubusercontent.com/79907936/110105234-2fd76880-7db9-11eb-99e2-81bfb300c22b.png)

Данная активность предоставляет пользователю возможность настроить основные свойства шарика. Все данные сохраняются в SharedPreferences, так что при повторном запуске приложения данные сохраняются.
